>setcargo

parametros: Cargo c, *Usuario(s) u

requer permissões: Gerenciar cargos

atribui o Cargo "c" para um ou mais Usuario(s) "u"
*se o parametro de Usuario(s) "u" for substituido por @everyone, todos os usuarios do servidor receberão o Cargo "c"

Uso:

  • >setcargo @admin @usuario1 @usuario2

    • atribui o cargo "admin" para o usuario1 e usuario2

  • >setcargo @admin @everyone

    • atribui o cargo "admin" para todos os usuarios do servidor




>rmcargo

parametros: Cargo c, *Usuario(s) u

requer permissões: Gerenciar cargos

remove o Cargo "c" de um ou mais Usuario(s) "u"
*se o parametro de Usuario "u" for substituido por @everyone, todos os usuarios do servidor perderão o Cargo "c"

Uso:

  • >rmcargo @admin @usuario1 @usuario2

    • remove o cargo "admin" do usuario1 e usuario2

  • >rmcargo @admin @everyone

    • remove o cargo "admin" de todos os usuarios do servidor

Uso:

>delmsg

parametros: Inteiro n

requer permissões: Administrador

deleta as ultimas "n" mensagens do canal de texto no qual o comando for usado

Uso:

>delmsgusr

parametros: Usuario u, Inteiro n

requer permissões: Administrador

no qual o comando for usado

Uso:

>ban

parametros: Usuario u, Inteiro t

requer permissões: Administrador

bane o Usuario "u" do servidor após "t" segundos
O segundo parametro é opicional

Uso:

  • >ban @usuario1 60

    • bane o usuario1 apos 1 minuto

  • >ban @usuario2

    • bane o usuario2 sem tempo de espera




>unban

parametros: Usuario u

requer permissões: Administrador

>queue
>q

parametros: *Texto t

requer permissões: nenhum

inicia uma pesquisa no youtube com o Texto "t" dado,
dos N resultados encontrados, o primeiro será adicionado a uma playlist de musica.

o bot então, caso não esteja conectado a nenhum canal de voz, irá entrar onde
o usuario estiver conectado, iniciando a playlist. Como se trata de uma playlist,
o uso subsequente desse comando irá adicionar as músicas em ordem, sendo que as
mesmas serão tocadas após o termino da atual.

*Texto "t" pode ser substituido por um link válido do youtube

Uso:

  • >queue musica legal

    • Pesquisa "musica legal" no youtube, caso sejam encontrados resultados,
    o primeiro irá ser tocado no canal de voz

  • >q musica ruim

    • Pesquisa "musica legal" no youtube, caso sejam encontrados resultados,
    o primeiro irá ser tocado no canal de voz
