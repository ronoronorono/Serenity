Comandos

>help

parametros: nenhum

requer permissões: nenhum

>ping

parametros: nenhum

requer permissões: nenhum

retorna o ping do usuario.



>setcargo

parametros: Cargo c, *Usuario(s) u

requer permissões: Gerenciar cargos

atribui o Cargo "c" para um ou mais Usuario(s) "u"
*se o parametro de Usuario(s) "u" for substituido por @everyone, todos os usuarios do servidor receberão o Cargo "c"

Uso:

  • >setcargo @admin @usuario1 @usuario2

    • atribui o cargo "admin" para o usuario1 e usuario2

  • >setcargo @admin @everyone

    • atribui o cargo "admin" para todos os usuarios do servidor



>rmcargo

parametros: Cargo c, *Usuario(s) u

requer permissões: Gerenciar cargos

remove o Cargo "c" de um ou mais Usuario(s) "u"
*se o parametro de Usuario "u" for substituido por @everyone, todos os usuarios do servidor perderão o Cargo "c"

Uso:

  • >rmcargo @admin @usuario1 @usuario2

    • remove o cargo "admin" do usuario1 e usuario2

  • >rmcargo @admin @everyone

    • remove o cargo "admin" de todos os usuarios do servidor



>spam

parametros: Usuario u, Inteiro n

requer permissões: Administrador

envia "n" mensagens privadas para o Usuario "u"

Uso:

  • >spam @usuario1 4

    • envia 4 mensagens privadas ao usuario1



>msgall

parametros: nenhum

requer permissões: Administrador

envia uma mensagem privada para todos os usuarios do servidor


>ban

parametros: Usuario u, Inteiro t

requer permissões: Administrador

bane o Usuario "u" do servidor após "t" segundos

Uso:

  • >ban @usuario1 60

    • bane o usuario1 apos 1 minuto
